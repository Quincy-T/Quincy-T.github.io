# Quincy Tordil - Portfolio Website

Personal portfolio website showcasing my academic journey, technical projects, and professional experience as a Computer Engineering student at Purdue University.

## Features

- **About**: Professional summary, education, work experience, skills, and leadership involvement
- **Projects**: Showcase of engineering and software projects
- **Courses**: Complete academic coursework organized by semester (newest to oldest)
- **Contact**: Ways to connect and get in touch

## Design

- Dark mode interface with gold and black color scheme
- Responsive layout for mobile and desktop viewing
- Interactive photo banner with collage animation
- Course tooltips displaying Purdue catalog descriptions

## Technologies

- HTML5
- CSS3
- JavaScript
- GitHub Pages for hosting
